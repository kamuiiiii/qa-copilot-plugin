# 首次初始化(空仓库 / 新环境引导)

> **何时读这份**:在一个仓库里**第一次跑**、或本次要测的**环境 / 站还没登记**时。把"建 target → 确认是否需要登录态 → 确认是否已登录 → connect 导出"串成一个有序闸门,补齐后才进 SKILL.md 的七步流程。
>
> 已经登记过 target、cookie 也验证过进站的仓库,**不用读这份**,直接走 SKILL.md「浏览器接入」默认路径。

## 触发条件

开跑前检测,命中任一条就先走本流程,**不要带着缺口闯进 step 1**:

- `targets/<env>.md` 不存在,或里面没有本次要测的站。
- 仓库根没有忽略 `secrets/` 与 `runs/` 的 `.gitignore`。
- (需要登录的站)`secrets/cookies-all.json` 不存在,或注入后验证仍停在登录入口。

逐项问清、补齐,**绝不自己编 URL、绝不替用户假设要不要登录**。

## 引导顺序

### 1. 建初始 target

和用户一起把本次要测的站登记进 `targets/<env>.md`(格式见 SKILL.md「输入:环境与站」的表):

- 问清:**环境名**(`qa` / `uat`)、站 **id**(短横线小写、跨环境同名)、**URL**、**说明**。
- 一个文件对应一个环境,一整组站登记在同一张 `## 站` 表里,每站一行 `| id | URL | 说明 |`。
- 顺手补 `.gitignore`:确保 `secrets/` 与 `runs/` 两行在里面(没有就就地创建 / 追加)。
- **仓库根没有 `CLAUDE.md` 就落一份薄指引**(已有则**不动、不覆盖**——那可能是用户自己的)。只写"仓库是什么 + 东西放哪 + 谁是源头",**一律指向 live 源,绝不复述七步流程 / AC 规范 / cookie 细节**(那些在 skill 里,复述会 stale)。模板:

  ```markdown
  # 这是一个 qa-copilot 产品仓库

  用于浏览器驱动的 QA 测试。**测试方法论由 `qa-copilot` skill 提供**(随插件安装)，
  直接用自然语言描述测试目标即可触发，本文件不复述流程。

  ## 目录
  - `targets/<env>.md` — 各环境的被测站清单(qa / uat …)，站 URL 的唯一来源
  - `secrets/cookies-all.json` — 登录态(gitignored，**绝不提交**)
  - `runs/<run-id>/` — 每轮测试产物(ac.md / 截图 / report.md)
  - `notes/<site>.md` — 跨 run 的导航 / 定位地图(可选)

  ## 起测
  描述要测的环境 + 站 + 目标，skill 会带你走"意图 → AC → 执行 → 报告"。
  首次在空仓库 / 新环境跑，skill 会先引导建 target、确认登录态、导出 cookie。
  ```

### 2. 确认是否需要登录态

问用户:**这个站要登录才能测吗?**

- **默认假设需要登录**(内部 QA 系统基本都要)——但必须显式问一次,让用户能否决。
- **不需要**(公开站 / 无鉴权页面)→ 跳过 cookie
- **需要** → 进第 3 步。

> 这一问是为了不给公开站强行套 cookie 注入。但**别自己猜**——拿不准就问,不要因为"看着像要登录"就默认跳过或默认导出。

### 3. 确认是否已登录 / cookie 是否就绪

仅当第 2 步判定"需要登录"时执行:

- `secrets/cookies-all.json` **不存在** → 走第 4 步导出。
- **存在** → 起一个 headless 实例注入并**验证是否真的进站**(复用 SKILL.md「默认路径」第 2 步:dump 正文,出现应用内真实数据 = 成功;仍只剩登录入口 = cookie 过期)。
  - 进站成功 → 初始化结束,进 step 1。
  - 仍停在登录入口 → 判定过期,走第 4 步重新导出。**不要**自己点 `Enter` 或试账号密码。

### 4. 连接并导出(A1)

cookie 不存在 / 已过期时执行:按 `cookie-export.md` 的 A1 流程重新导出 cookie。导出后起 headless 注入并**再验证一次进站成功**(同第 3 步),确认通过才算初始化完成,进 step 1。

## 完成判定

初始化算完成,当且仅当:

- 本次要测的站已登记进 `targets/<env>.md`,URL 来自用户、非编造;
- `.gitignore` 已忽略 `secrets/` 与 `runs/`;
- 要么用户确认本站**无需登录**,要么 cookie 已导出且 headless 注入**验证进站成功**。

满足后回到 SKILL.md `## 工作流总览` 的 step 1(解析意图)继续。
